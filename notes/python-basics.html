Chapter 6: Operators - The Tools of Programming

üë¶
Alex

üëß

print(10 - 3)   # Subtraction: 7

print(10 / 3)   # Division: 3.333...

üë¶
Alex

üëß

Let's explore all the operator families: 
# 2. ASSIGNMENT OPERATORS (Storing values)

x -= 3       # Same as x = x - 3 (now 12)
x *= 2       # Same as x = x * 2 (now 24)
x /= 4       # Same as x = x / 4 (now 6.0)

# 3. COMPARISON OPERATORS (Making comparisons)
print(5 == 5)   # Equal to: True
print(5 != 3)   # Not equal: True
print(5 > 3)    # Greater than: True
print(5 < 3)    # Less than: False
print(5 >= 5)   # Greater or equal: True
print(3 <= 5)   # Less or equal: True

# 4. LOGICAL OPERATORS (Combining conditions)
age = 15
has_permission = True
print(age > 13 and has_permission)  # Both must be true
print(age > 18 or has_permission)   # At least one true
print(not (age > 18))               # Opposite

üéØ Most Used Operators

  • Math: + - * / // % **
  • Compare: == != > < >= <=
  • Logic: and, or, not
  • Check: in, not in

üîß Special Operators

  • Identity: is, is not
  • Membership: in, not in
  • Bitwise: & | ^ ~ << >>
  • (Advanced - learn later!)
Quick Reference! Focus on arithmetic, comparison, and logical operators first. You'll use these 90% of the time. The others are for special situations!
